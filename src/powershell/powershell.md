# Info about "powershell" part of this repo
* Developed with Powershell Core `7.4.6` on Windows (AMD64)
* To run the program, via Powershell terminal:
  ```powershell
  $ .\$day.ps1
  ```
  or via Bash terminal:
  ```sh
  $ pwsh ./$day.ps1
  ```
